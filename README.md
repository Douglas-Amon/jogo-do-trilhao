# 🎯 Show do Trilhão

Jogo educativo interativo inspirado no clássico programa de perguntas e respostas, desenvolvido com **JavaScript e p5.js**.

## 🧠 Sobre o projeto

O **Show do Trilhão** é um jogo que tem como objetivo testar e reforçar conhecimentos de forma divertida. O jogador escolhe um tema e responde perguntas para acumular dinheiro fictício.

* ✔️ Acertou → ganha dinheiro
* ❌ Errou → o jogo termina
* 🏆 Finalizou o tema → vitória

---

## 📚 Temas disponíveis

* Português
* Matemática
* Ciências
* História
* Geografia
* Variedades

---

## 🚀 Funcionalidades

* 🎮 Interface interativa com p5.js
* 🎲 Perguntas embaralhadas automaticamente
* 💰 Sistema de pontuação (R$1000 por acerto)
* ❌ Tela de derrota ao errar
* 🏆 Tela de vitória ao concluir o tema
* 🔊 Sons inspirados no Show do Milhão
* 📖 Tela de instruções e créditos

---

## 🛠️ Tecnologias utilizadas

* JavaScript
* p5.js

---

## ▶️ Como executar

1. Acesse o projeto no p5.js ou baixe os arquivos
2. Abra o arquivo `index.html` no navegador
   ou
3. Execute diretamente no editor online: https://editor.p5js.org/
